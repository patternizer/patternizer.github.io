Talks thumbnails

Aspect ratio: 16:9 (the CSS enforces aspect-ratio: 16/9; object-fit: cover)
Max on-page width (desktop, 2-up layout): ~430 px wide
Recommended source size (2× “retina”): 860×484 (round to 960×540 or 1280×720 if that’s easier)
Format & size tips: Use WebP (or JPEG if needed), aim for ≤150 KB. Keep key content away from the edges (it’s crop-safe due to object-fit: cover).

Publications thumbnails

Aspect ratio: 4:3 (CSS fixes width:120px; height:90px)
On-page size: 120×90 px (100×75 px on narrow screens)
Recommended source size (2× “retina”): 240×180 px (3× optional: 360×270)
Format & size tips: Logos/diagrams → PNG/WebP; photos → WebP/JPEG. Aim for ≤30 KB.
